Infusion Pump

Universal device for all infusions and transfusions Accurate & Work with all brands of infusion tube Accuracy Error <±5% after calibrated, work with all brands of IV tube: 10d/ml, 15d/ml, 20d/ml, 60d/ml (diameter 3.4mm-4.5mm) Safety and Reliable 2 CPU systems for self-detection; Multiple sensors can detect and monitor the infusion status Multi-functions 12 infusion mode including Programmable infusion mode; Drug Library and history records data can be transferred; WIFI with communication Interface for information system; Warmer for heating.

    Specifications of Infusion pump:

    Model Single-channel infusion pump WIT-601A
    Communication (Optional) WIFI, with communication protocol interface for information system
    Warmer (Optional) Warmer on the side of pump to keep the medical liquid warm
    Drug Library Library size configurable: More than 1000 drugs & 54 drug dose units
    History Records Up to 20,000 infusion records
    Data Transfer RS485 port/compatible with RS232, Wired interface connectable with PC to upload & download data
    Display High brightness color LCD 3.2-inch screen, Adjustable brightness
    Infusion Mode Rate/VTBI, Time/VTBI, Rate/Time, Volume, Drop, Micro, Simple Rate, Programmable, DOSE Weight, Drag lib, Auto Bolus, Manual Bolus
    Programmable 8 programmable sequences of "Time & Flow rate & VTBI"
    IV SET 10d/mL, 15d/mL, 20d/mL, 60d/mL (diameter = 3.4mm - 4.5mm). Open system for all brands of IV tube
    Accuracy Error More than 50 bands if V tube feature data can be stored inside the pump
    Calibration Volume Mode: < ±5%, Drop Mode: < ±2%
    Flow Rate Automatic or manual calibration
    Flow Rate Increment Volume: 0.1nm/h~1200mh/h, Drop: 1d/min~400d/min
    VTBI 0.01m/h
    Total Volume 0.1~9999.9 ml
    KVO 0.1~9999.9 ml
    Purgs Default: 1m/h; 8.1m/h~20mh/h can be set
    Bolus Keep pushing down "Bolus" button to run the default Bolus parameters. Or click the "Bolus" button to enter the Bolus parameter editable interface
    Upstream Pressure High / Middle / Low, or OFF
    Downstream Pressure 20 pressure Levels: 200mmHg; 1200mmHg (mmHg & kPa)
    Bubble Detected by ultrasonic sensor with adjustable 5 levels, 50μL ~ 90μL
    Drop Sensor Infrared sensor to detect the drops, monitoring the infusion status
    Alarms Audio and Visual alarm for: Bubble, Door open, Finish, Pause timeout, Doom Occlusion, Up Occlusion, Lower Battery of damage, Motor error, Memory X Set Prog, Pressure test failure, Motor failure, Abnormal drop rate, Heater failure and etc. 30 kinds safety-monitoring alarm
    Self-detection 2 CPU systems; Self-detection at startup and running time
    Non-interrupt Parameters can be modified when pump running
    Keypad Lock Automatically or Naturally lock the keyboard
    AC Power AC100V ~ 240V, 50Hz/60Hz, 25Wx. Vehicle-mounted power supply adapter (Optional)
    Battery Ni-MH battery, DCS 0K, 2000mAh (6 hours at 25m/h rate)
    Electrical Safety Class Class 1, Type CF, IPX4
    Install Can be used either on IV stand, mile or as a table top device
    Weight & Dimension 2.0kg; 130mm x 120mm x 195mm
    Working Environment 5°C ~ 40°C; 20% ~ 90% humidity; 70kPa ~ 108kPa atmospheric
    Application Medical locations such as intensive care unit, pediatric ward, obstetrics and provoking department for computerized autonomous infusion of drugs
