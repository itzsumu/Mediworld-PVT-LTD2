Syringe Pump

Accurate & Work with all brands of syringe Accuracy Error <±2% after calibrated, work with all brands of 5ml, 10ml, 20ml, 30ml, 50ml (60ml) syringes which can be automatically sensed Safety and Reliable 2 CPU systems for self-detection; Multiple sensors can detect and monitor the injection status Multi-functions 10 injection mode including Programmable injection mode; Drug Library and history records data can be transferred; WIFI with communication Interface for information system

    The Specifications of Syringe pump:
    Model Single-channel syringe pump WIT-301A
    Communication (Optional) WIFI, with communication protocol interface for information system
    Drug Library Library size configurable; More than 1000 drugs & 54 drug dose units
    History Records Up to 20,000 infusion records
    Data Transfer RS485 port/compatible with RS232, Wired interface connectable with PC to upload & download data
    Display High brightness color LCD 3.2-inch screen, Adjustable brightness
    Injection Mode Rate/YTBt, Time/YTBt, Rate/Time, Micro, Simple Rate, Programmable, DOSE Weight, Drug Lib, Auto Bolus, Manual Bolus
    Programmable 8 programmable sequences of "Time & Flow rate & YTB"
    Syringe 5ml, 10ml, 20ml, 30ml, 50ml(60ml) syringes can be used and automatically identified. Open system for all brands of syringes. More than 50 brands syringes feature data can be stored inside the pump
    Accuracy Error < ±2%
    Calibration Automatic or manual calibration
    Flow Rate
    • 5ml syringe: 0.10-150 ml/h
    • 10ml syringe: 0.10-300 ml/h
    • 20ml syringe: 0.10-400 ml/h
    • 30ml syringe: 0.10-800 ml/h
    • 50ml (60ml) syringe: 0.10-1200 ml/h
    Flow Rate Increment 0.01 ml/h
    YTBI 0.1-9999.9 ml
    Total Volume 0.1-9999.9 ml
    KVO Default: 1ml/h; 0.1ml/h~20ml/h can be set
    Bolus Keep pushing down "Bolus" button to run the default Bolus parameters. Or click the "Bolus" button to enter the Bolus parameter editable interface
    Anti-Bolus Automatically back off to release the pressure of occlusion to avoid risk of "Bolus"
    Occlusion Pressure 20 pressure levels: 200mmHg; 1200mmHg (mmHg & kPa)
    Alarms Audio and Visual alarm for: Occlusion, Occlusion drop rapidly, Near Finish, Finish, KVO, Pause timeout, Pauber error, Compression bar error, Low Battery or Battery Discharged, Motor error, Parameter error, Interrupt of AC Power etc
    DEES Automatically check and adjust to eliminate the mechanical gap to reduce dose error at startup
    Self-detection 2 CPU systems; Self-detection at startup and running time
    Non-Interrupt Parameters can be modified when pump running
    Keypad Lock Automatically or Manually lock the keyboard
    AC Power AC100V ~ 240V, 50Hz/60Hz, 25Wx. Vehicle-mounted power supply adapter (Optional)
    Battery Hi-MH battery, DC36V, 2000mAh (7 hours at 5ml/h rate)
    Electrical Safety Class Class 1, Type CF, IPX4
    Install Can be used either on IV stand, rails or as a table top device; Multiple pumps can be superimposed up and down
    Weight & Dimension 1.9kg; 140mm x 145mm x 325mm
    Working Environment 5°C ~ 40°C; 20% ~ 90% humidity; 70kPa ~ 108kPa atmospheric
    Application Medical locations such as extensive care unit, pediatric ward, obstetrics and gynecology department for constant speed intravenous infusion of drugs
